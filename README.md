# homebrew-tap

Homebrew tap for [Lionel Panhaleux](https://github.com/lionel-panhaleux)'s tools.

```sh
brew install lionel-panhaleux/tap/pst
```

- [`pst`](https://github.com/lionel-panhaleux/pst) — plain-simple-tickets: a git-friendly,
  one-line-per-ticket work tracker for AI agents. Built from source (no prebuilt binaries).
